Capítulo 1: ¿Qué es una red?

Definición

Una red de computadoras es un conjunto de dispositivos interconectados que pueden intercambiar información y compartir recursos mediante protocolos de comunicación.

Estos dispositivos pueden ser ordenadores, servidores, teléfonos móviles, impresoras, routers, switches o cualquier equipo capaz de enviar y recibir datos.

El objetivo principal de una red es permitir la comunicación eficiente, segura y confiable entre distintos sistemas.

Elementos básicos de una red

Toda red está compuesta por varios elementos fundamentales:

Dispositivos finales (hosts)

Son los equipos que generan o consumen información. Ejemplos: ordenadores, servidores, smartphones, impresoras.

Dispositivos intermedios

Son los equipos que permiten que la comunicación ocurra. Ejemplos: routers, switches, puntos de acceso WiFi.

Medios de transmisión

Es el canal físico o inalámbrico por donde viajan los datos. Puede ser cable de cobre, fibra óptica o señales de radio.

Protocolos

Son las reglas que determinan cómo se envía y recibe la información. Ejemplos: TCP/IP, HTTP, DNS.

Modelo Cliente-Servidor y Peer-to-Peer (P2P)

Modelo Cliente-Servidor

En este modelo, un equipo llamado servidor ofrece servicios o recursos, mientras que uno o varios clientes solicitan dichos servicios.

Ejemplo: cuando accedemos a una página web, nuestro navegador actúa como cliente y el servidor web responde con el contenido solicitado.

Modelo Peer-to-Peer (P2P)

En el modelo P2P no existe un servidor central. Todos los dispositivos pueden actuar como cliente y como servidor al mismo tiempo.

Ejemplo: redes de intercambio de archivos o aplicaciones descentralizadas.

Tipos de redes

LAN (Local Area Network)

Red de área local. Cubre un espacio reducido como una casa, oficina o edificio. Es el tipo de red más común.

MAN (Metropolitan Area Network)

Red de área metropolitana. Conecta múltiples LAN dentro de una ciudad.

WAN (Wide Area Network)

Red de área amplia. Conecta redes a gran escala, incluso entre países o continentes. Internet es el ejemplo más grande de una WAN.
