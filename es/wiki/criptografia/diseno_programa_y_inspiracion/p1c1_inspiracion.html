# Inspiración Si quisiéramos crear un juego o reto criptográfico o algo similar, como fuentes de inspiración podríamos recurrir a las siguientes ideas: - Componer nuestros propios textos, especial mención a ideas o pensamientos filosóficos. - Crear un poema. - Referenciar o usar una obra de otro artista, como un texto literario o la descripción de una de sus obras. - Texto arqueológicos, históricos, científicos... importantes. Aunque no tienen que ser conocidos globalmente. Podemos usar archivos o registros históricos públicos para recopilar información o ideas como el Vaticano, Simancas, universidades, ayuntamientos, museos... - Recurrir a textos desclasificados de espionaje, militar o gubernamentales. - Destacar los dobles significaos de algunas palabras, su poder simbolico. Por último, decir que por texto entiéndase cualquier tipo de información, o referencia a esta, como un mapa, una partitura...
