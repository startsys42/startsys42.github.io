Capitulo 1: Conceptos criptograficos

¿Qué es la criptografía?

La criptografía es la disciplina que estudia técnicas para proteger la información. Su objetivo principal es garantizar la seguridad de los datos mediante tres propiedades fundamentales:

\begin{itemize} \item \textbf{Confidencialidad:} Solo las personas autorizadas pueden acceder a la información. Por ejemplo, cuando envías un mensaje cifrado, nadie fuera del destinatario puede leerlo. \item \textbf{Integridad:} La información no debe ser alterada de manera no autorizada. Si un mensaje se modifica, el receptor puede detectarlo. \item \textbf{Autenticidad:} Permite verificar que la información proviene realmente de quien dice ser el remitente. Esto evita suplantaciones. \end{itemize}

A mayores se desea cumplir las siguientes dos propiedades:

\begin{itemize} \item \textbf{Disponibilidad:} Los datos y servicios deben estar accesibles cuando se necesiten. \item \textbf{No repudio:} Garantiza que quien envió la información no pueda negar haberla enviado, proporcionando evidencia de su origen. \end{itemize}

La criptografía se utiliza para asegurar comunicaciones, transacciones electrónicas y almacenamiento seguro de datos.
